# memo_ts

## 项目概况
```
    本项目基于vue2.0开发，使用tyscript语法编写。
    根据本地的localstorage数据存储功能，实现日记的新增及记录功能。
```

### 开发技术栈
```
    vue 2.x
    vue-property-decorator 8.4.x
    element-ui 2.4.x
```

### 新建项目
```
    vue create memo_ts
    cd memo_ts
    vue add @vue/typescript
    vue add element
```

### Lints and fixes files
